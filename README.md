## Описание

Расширение для Visual Studio Code, которое упрощает работу с комментариями вида `TODO`, `FIXME` и `BUG` в исходном коде.

Плагин автоматически:

- находит все TODO/FIXME/BUG в текущем открытом файле;
- подсвечивает соответствующие строки в редакторе;
- отображает список всех найденных комментариев в отдельной панели **TODO Highlighter** в боковой панели (TreeView);
- позволяет переходить к нужному TODO по клику на элементе списка.

---

## Возможности

- Автоматический поиск строк, содержащих TODO, FIXME или BUG.

- Подсветка найденных строк в активном файле.

- Отображение всех найденных пометок в отдельной панели слева («TODO Highlighter»).

- Быстрый переход к нужной строке кода по клику на элемент списка.

- Автоматическое обновление при любом изменении текста.

- Поддержка всех языков программирования (анализ идёт по тексту, а не по синтаксису).

---

## Применение

Этот плагин удобен при разработке для:

- отслеживания важных мест в коде,

- анализа технического долга,

- улучшения навигации по проекту,

- быстрого поиска проблемных участков.
